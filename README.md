## Lo que aprendí

Practiqué cómo construir un layout asimétrico con CSS Grid usando `grid-template-areas` en dos breakpoints distintos, reubicando las tarjetas sin tocar el HTML. También usé selectores agrupados para aplicar los mismos colores a tarjetas con esquemas visuales similares sin repetir código.
